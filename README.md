# Tesis-tamano-efecto-bigdata

Códigos completos en R complementarios de la tesis “El tamaño del efecto como medida de significancia en Big Data”.

## Estructura
- R/: scripts por capítulo y medida (cada carpeta contiene un archivo.R ejecutable).
- data/: datos simulados (opcional).
- output/: no se incluyen figuras en el repositorio; los scripts pueden generarlas si se activa la opción de guardado.
